What is Veins of the Earth?

Veins of the Earth is a free open source roguelike with RPG elements, in development since 2012.

The RPG ruleset was originally conceived based on the d20 dice. Currently we use d100.

Versions

The project has gone through several rewrites, in different programming languages and libraries.

All versions are open source and available from Github repositories.

The only version that was actually distributed more widely was the original iteration, made in T-Engine - ModDB link.

In fact, distribution problems are a main reason why the current iteration is a web game.

System requirements

Depending on the version, they differ. Most versions should run on just about anything, provided you have the IDE for the programming language used, and the source code.

T-Engine requires some OpenGL support and will not run on old video cards.

Similarly, the current Godot iteration requires OpenGL 3.
